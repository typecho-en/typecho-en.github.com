Typecho Documentation

  • How to install

    • It have just few differences from most blog script. Multi-database mode such as SQlite completely support make it simpler and faster. Do 3 steps in Installer, you can enjoy in Typecho powerful and completely experience within 5 minutes.

    If you have problems during installing, you can go to our community or mail list to ask for help anytime.

    MySQL Install Mode

    Most of people use MySQL Database, so there is a simple installation instrution for MySQL, you can do like this in other database.

    Download the latest version and uncompress it

    Browse http://typecho.org/download get latest stable version, after installing uncompress it, you will see these directories and files:

    1. admin directory
    2. install directory
    3. usr directory
    4. var directory
    5. license.txt file
    6. index.php file
    7. install.php file

    Upload to specified directory on server

    Upload them to specified directory on server such as root directory or /blog, or the others you want.

    Browse the installation files directory

    After uploading, use browser to go into the directory, and you will see Typecho Installer. If you don't see red warning text at the top, your server fully supports Typecho, click to next step.

    Fill your database information

    Choose MySQL Original Function Adapter in the adapter, the next two fields don't need to change unless you have special needs. Go to the third fiels to fill your database information.

    We don't suggest to change database prefix to make compatibility of upgrading and plugin

    Fill Website Administrator Information

    Installer will detect your url automatically, if you use subdomain such as blog.xx.com, please change by yourself. Username can be diy for you. E-Mail uses for Gravatar.

    For now, configation initizing is ready, if you don't make mistakes, click next, then finish Typecho Install in a minute.

    Finish Installing

    In the install success page will show the login password generated automatically, please remember it and follow the tips to change. All things are done, have fun with Typecho! :)

    If your initial password is lost, you can delete config.inc.php file automatically generated in the install directory, then choose to leave original data to reinstall
